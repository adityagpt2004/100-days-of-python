# Day 01 - Band Name Generator

This is the first project from the **100 Days of Python Bootcamp by Dr. Angela Yu**.

## Project Description
The Band Name Generator is a simple Python program that creates a fun band name based on user input.

The program asks the user for:
- The city they grew up in
- The name of their pet

It then combines these inputs to generate a unique band name.

## Example Run
Welcome to the Band Name Generator.
What's the name of the city you grew up in?
Delhi
What's your pet's name?
Bruno
Your band name could be Delhi Bruno


## Concepts Used

- Python `input()` function
- String concatenation
- Variables
- Basic user interaction
